# Cafe Restaurant Frontend Website

A modern and responsive restaurant website built using Next.js, React, and Tailwind CSS. This project focuses on clean UI design, component-based architecture, and optimized user experience.

## 🚀 Features
- Responsive design (mobile, tablet, desktop)
- Modern and clean user interface
- Reusable components structure
- Fast performance with Next.js
- Styled using Tailwind CSS

## 🛠️ Technologies Used
- Next.js
- React
- Tailwind CSS

## 📌 Project Purpose
This project was developed to strengthen frontend development skills, focusing on building responsive layouts, reusable components, and improving UI/UX design practices.

## ▶️ How to Run Locally
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
3.Run the development server: npm run dev

4.Open http://localhost:3000 in your browser
